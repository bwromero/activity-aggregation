# Screenshots

Add your application screenshots here:

1. **dashboard.png** - Main dashboard view showing the application interface
2. **data-table.png** - Data table with filters and aggregation controls

## How to Take Screenshots

1. Run your application (both frontend and backend)
2. Navigate to `http://localhost:4200`
3. Take a screenshot of the main view → save as `dashboard.png`
4. Apply some filters (check Project, Employee, Date) → save as `data-table.png`
5. Screenshots will automatically appear in the main README.md!

## Recommended Tools

- **Windows:** Snipping Tool (`Win + Shift + S`)
- **Mac:** Screenshot (`Cmd + Shift + 4`)
- **Linux:** Screenshot or `gnome-screenshot`
